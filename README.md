# 💻 Web 开发作业 2

## HTML、PHP 登录系统与数据库管理

---

## 📌 作业说明

本作业旨在评估学生在 **HTML 基础、PHP 用户认证系统以及 MySQL 数据库操作** 方面的能力。

---

## 💻 使用技术

* HTML
* PHP（仅限原生 PHP，不可使用框架）
* MySQL（XAMPP 或 MySQL Workbench）

---

## 📚 作业任务

---

### 🧾 任务 1：HTML 基础与文本操作（20 分）

请编写 HTML 代码显示以下文字：

> **"Hello World"**

然后修改代码，实现以下要求：

#### ✅ 要求：

* 修改显示内容为：

  > **"I am from Synergy College"**
* 手动调整字体大小
* 修改文字颜色
* 使用正确的 HTML 结构

---

### 🔐 任务 2：用户注册与登录系统（40 分）

使用 HTML 和 PHP 开发一个简单的 **注册与登录系统**。

#### 🖥️ 前端要求：

* 创建输入栏：

  * 用户名（Username）
  * 密码（Password）
* 包含按钮：

  * 注册（Register）
  * 登录（Login）

---

#### ⚙️ 后端要求：

##### 📝 注册功能：

* 使用 PHP：

  * 获取表单输入数据
  * 将用户名和密码存入 MySQL 数据库

##### 🔑 登录功能：

* 使用 PHP：

  * 验证用户输入
  * 检查账号与密码是否与数据库匹配

---

#### ⚠️ 注意：

* 不要求使用 CSS
* 数据必须存储在数据库中（不可硬编码）

---

### 🗄️ 任务 3：数据库管理与 SQL 语句（40 分）

请使用 **XAMPP（MySQL）** 或 **MySQL Workbench** 完成以下操作。

#### 📊 表格要求：

创建一个包含 **5 个字段及至少 6 条数据** 的数据表。

#### 🧱 字段名称：

* 收货日期（Receiving Date）
* 运单号（Tracking Number）
* 产品名称（Product Name）
* 体积（CBM）
* 重量（Weight）

---

#### ⚙️ SQL 操作要求：

##### ✅ 数据操作：

* 插入数据（INSERT）
* 修改数据（UPDATE）
* 删除数据（DELETE）

##### 🔍 数据查询：

* 使用条件筛选（WHERE）
* 数据排序（ORDER BY）

##### 🏗️ 表结构修改：

* 添加字段（ADD COLUMN）
* 删除字段（DROP COLUMN）
* 修改表结构

---
